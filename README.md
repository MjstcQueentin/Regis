# Régis

Régis est un logiciel développé pour [Drama Tea'c Circus](https://linktr.ee/dramateaccircus), pour le lancement de sons et de vidéos sur vidéoprojecteur pendant des représentations théâtrales.

Il est conçu pour être utilisable avec le matériel le plus rudimentaire, seul un navigateur Web à jour est requis pour son fonctionnement. Les sons et vidéos sont lancées via les APIs du navigateur, selon les codecs pris en charge par le système.
